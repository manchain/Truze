# Supabase Setup Instructions

1.  **Create a Project**: Go to [Supabase](https://supabase.com/) and create a new project.
2.  **Get Credentials**:
    *   Find the **Project URL** and **anon public key**.
    *   Add them to your `.env` file (copy content from `.env.example` first if needed).

    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_project_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
    ```

3.  **Create Table**: Run the following SQL in the Supabase SQL Editor to create the `posts` table:

    ```sql
    create table public.posts (
      id bigint generated by default as identity not null,
      created_at timestamp with time zone not null default now(),
      author text null,
      author_address text null,
      avatar text null,
      timestamp timestamp with time zone null,
      title text null,
      content text null,
      tags text[] null,
      likes bigint null default '0'::bigint,
      comments bigint null default '0'::bigint,
      status text null,
      verification_count bigint null default '0'::bigint,
      constraint posts_pkey primary key (id)
    ) tablespace pg_default;
    ```
